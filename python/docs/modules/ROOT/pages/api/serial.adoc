= Serial

This page contains a listing of the `pypalmsens.serial` module.

For example, to load your custom CV experiment from a `.mscr` file:

[,python]
----
>>> import pypalmsens as ps

>>> path = 'custom_cv.mscr'

>>> port = methodscript.auto_detect_port()

>>> with ps.serial.Serial(port, 1) as comm:
...     device = ps.serial.Instrument(comm)
...     device_type = device.get_device_type()
...     print(device_type)
...     device.send_script(path)
...     result_lines = device.readlines_until_end()

>>> curves = ps.serial.parse_result_lines(result_lines)
----
