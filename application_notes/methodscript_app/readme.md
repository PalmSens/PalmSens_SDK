# From Methodscript to Maintainable Python Application

The code here accompanies the application note: [From Methodscript to Maintainable Python Application](https://www.palmsens.com/knowledgebase-article/xxx/).

The application note demonstrates how to take an optimized methodscript,
and turning it into a application in production. The application note includes a example dashboard for configuring the method and viewing the data.

There are 2 scripts:
- [cv.mscr](./cv.mscr): This is the methodscript generated by PSTrace.
- [app.py](./app.py): An example [Streamlit](https://docs.streamlit.io) dashboard for starting a measurement and viewing the data.

See the application note for instructions.

## Running the dashboard locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
